# TreeTab

**以看板形式管理你的浏览器标签页**

TreeTab 是一个 Chrome 扩展，将新标签页替换为一个直观的标签管理看板：
- **上方**：按域名分组的标签页（自动聚合相同域名，支持重复检测、一键清理）
- **下方**：Chrome 原生标签组（横向卡片布局，支持拖拽管理、重命名、一键关闭）

无服务器，无账户，无外部 API 调用。纯 Chrome 扩展。

---

## 快速安装

### 方式一：编码 Agent 安装

将本仓库发给你的编码 Agent（Claude Code、Codex 等），发送：**"install this"**：

```
https://github.com/userAlex110/TreeTab
```

Agent 会带你完成安装，约 1 分钟。

### 方式二：手动安装

**1. 克隆仓库**

```bash
git clone https://github.com/userAlex110/TreeTab.git
```

**2. 打开扩展管理页面**

在 Chrome 地址栏输入以下地址并回车：

```
chrome://extensions
```

你会看到已安装的扩展列表页面。

**3. 开启开发者模式**

在扩展管理页面的 **右上角**，找到 **"开发者模式"** 开关，点击启用。开启后页面顶部会出现一行新的操作按钮。

**4. 加载扩展**

点击页面左上角的 **"加载已解压的扩展程序"** 按钮，会弹出文件夹选择对话框。导航到你刚才克隆的仓库，选择其中的 **`extension/`** 文件夹（注意是 `extension` 子文件夹，不是整个仓库根目录），然后点击 **"选择"**。

> **提示**：如果你是下载的 ZIP 压缩包，请先解压，然后在解压后的目录中找到 `extension/` 文件夹。

**5. 验证安装**

加载成功后，扩展列表中会出现 **TreeTab**，状态显示为"已启用"。此时打开一个新标签页（`Ctrl/Cmd + T`），你将看到 TreeTab 看板界面。

> **常见问题**：如果新标签页没有变化，请确认 TreeTab 扩展的开关是开启状态，然后重新打开一个新标签页。

## 功能特性

### 上方区域 - 域名分组

- **按域名自动分组** 所有标签页
- **主页特殊分组** - Gmail、X、YouTube、LinkedIn、GitHub 主页聚合
- **重复检测** - 高亮显示重复页面，一键清理
- **展开查看更多** - 每组显示前 8 个，点击 "+N more" 展开
- **拖拽到分组** - 将标签页拖入上方分组或新建分组

### 下方区域 - 标签组管理

- **显示 Chrome 原生标签组**，按横向卡片布局
- **拖拽管理** - 将标签页拖入分组，或在分组间移动
- **重命名分组** - 点击组名即可编辑
- **一键关闭组内所有标签页**
- **未分组标签** - 独立卡片显示，支持拖入分组

### 核心功能

| 功能 | 描述 |
|------|------|
| 🔍 重复检测 | 自动识别相同页面，显示 (2x) 标记，一键去重 |
| 🎯 快速跳转 | 点击任意标签直接切换到该标签页 |
| ✨ 关闭动效 | 满足感的滑过动画 + 彩纸爆发 + 音效 |
| 🏷️ 自定义标题 | 针对特定系统（如智慧教育）显示友好标题 |
| 📊 徽章计数 | 工具栏徽章显示标签数量，颜色编码健康度 |
| 🔒 100% 本地 | 数据存储在 `chrome.storage.local`，永不离开设备 |

## 常见问题

### 为什么看不到我的标签组？

标签组是 Chrome 的原生功能。如果你没有创建过标签组，上方区域会显示"未分组"卡片。你可以：
1. 在 Chrome 中右键标签 → "将标签页添加到新组"
2. 或者从下方拖拽标签到"新建分组"区域

### 如何退出 TreeTab？

TreeTab 替换的是新标签页。要恢复正常，只需：
1. 在地址栏输入任意网址（如 `chrome://extensions`）
2. 或者按 `Cmd/Ctrl + T` 后在地址栏输入网址

### 数据会上传吗？

不会。所有数据存储在 `chrome.storage.local`，完全本地运行。
